# Open source Gemini client for iOS
> Created because the official Google app does not support iOS 15, and I needed Gemini on my iPhone 6S.

Features:
* Written entirely in SwiftUI
* Markdown support
* System prompts (can be saved)
* Temperature control
* Support for images and files
* History

To get started, you only need a Google AI Studio API key, which is free to generate.
